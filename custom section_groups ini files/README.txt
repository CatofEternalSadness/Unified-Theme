######################################## [ INSTALLATION ] ########################################

These ini files are preconfigured to work on the Adam image for the RG350/280 devices.

Using Simplemenu 10 or greater:

	Back up your original ini files, edit the files as needed, then simply copy these files into the: /media/data/local/.simplemenu/section_groups		folder replacing any files that already exist.

Using Simplemenu 9.4 or lower: (just update if possible)

	Move these files into the: Unified/resources/GENERAL/section_groups	folder.

########################################### [ NOTICE ] ###########################################

While these ini files are completely optional to use, these files will add a few extra sections, as well as reorganize the systems for a better experience.

Sections added:

ATARI 800
ATARI ST
TURBOGRAFX-16
TURBOGRAFX-CD
NEO GEO POCKET COLOR
WONDERSWAN COLOR
MSX2
CP SYSTEM 2
RPG MAKER*

The folders needed for these systems will be included in the roms folder.

*The EasyRPG opk is not included with the Adam image or this theme and must be manually added.

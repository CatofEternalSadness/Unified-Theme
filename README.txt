######################################## [ INSTALLATION ] ########################################

Put the Unified folder in: /media/data/local/home/.simplemenu/themes/320x240/

########################################### [ NOTICE ] ###########################################

This theme was not designed to be used with game art. For the best experience possible, use Simplemenu 9.5 or greater.

	If you are using Simplemenu 9.4 or lower, make sure you do not have game art in the media folders of your roms directory.

This theme uses a custom section_groups file. (ALWAYS BACKUP YOUR FILES BEFORE CHANGING THEM)
	
	Either edit the .ini files in: Unified/section_groups/ that come with the Unified theme,
	or copy over your existing .ini files in: /media/data/local/home/.simplemenu/section_groups/ to the: Unified/section_groups/ folder.
